## Outils

### Éditeur de texte simplifié

Coller dans l'URL du navigateur :

```html
data:text/html,%20%3Ctitle%3ETexte%3C/title%3E%3Cbody%20contenteditable%20style=%22font:%202rem/1.5%20monospace;max-width:60rem;margin:0%20auto;padding:4rem;%22%3E
```

Mode sombre :

```html
data:text/html,%20%3Ctitle%3ETexte%3C/title%3E%3Cbody%20contenteditable%20style=%22font:%202rem/1.5%20monospace;max-width:60rem;margin:0%20auto;padding:4rem;background-color:%23001f3f;color:white;%22%3E
```
