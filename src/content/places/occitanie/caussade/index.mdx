Caussade est une petite ville tranquille, qui n'a rien de spécial, mais c'est un bon pied-à-terre si vous comptez rester un jour ou deux pour faire des randonnées et visiter les villages environnants comme Saint-Antonin-Noble-Val. Ici, la voiture est essentielle pour bien profiter des environs.

Arrangez-vous pour y être le jour du marché. Il s'étend dans toutes les petites rues, transformant la ville ; ça vaut vraiment le détour !
